# Stellar Privacy Mixer GitHub Pages site

Static landing page for the `stellar-mixer` GitHub Pages site.

## Files

- `index.html`
- `styles.css`
- `script.js`
- `assets/logo.png`
- `assets/desktop-app.png`
- `.nojekyll`

No build step is required.

## Local preview

Open `index.html` directly in a browser, or run:

```bash
python3 -m http.server 8080
```

Then open:

```text
http://localhost:8080
```

## Deploy target

Organization GitHub Pages repository:

```text
stellar-mixer/stellar-mixer.github.io
```

After pushing to `main`, GitHub Pages should publish the site at:

```text
https://stellar-mixer.github.io
```
